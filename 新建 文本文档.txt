## ✅ 方案2：【极简无门槛】ttkbootstrap (最快！不用改框架，适合新手，颜值高)
### ✅ 适用人群
如果你现在的GUI是用 **Python自带的 tkinter/ttk** 写的（大概率是这个，因为tkinter最常用，也最容易写丑），这个方案**0学习成本、0重构代码、最快上手**，**10行代码就能把丑版tkinter变成高颜值界面**，绝对的懒人福音！

### ✅ 适配你的GUI组件
✅ 按钮 ✅ 输入框 ✅ 下拉选择器 ✅ 列表框 ✅ 复选框 ✅ 标签页 ✅ 菜单栏 ✅ 表格
### ✅ 优点
1. 改代码最少：原生tkinter代码，**只需要改1行导入、1行主题设置**，其他所有组件逻辑完全不变；
2. 体积最小：打包exe只多1M，几乎无感知；

### ✅ 完整可运行示例代码（复制即用）
```python
import tkinter as tk
import ttkbootstrap as ttk
from ttkbootstrap.constants import *
from ttkbootstrap.scrolled import ScrolledListbox

# 1行替换：把tk.Tk()换成ttk.Window()，指定深色主题
window = ttk.Window(title="金融分析系统 - ttkbootstrap风格", themename="darkly") # 主题可选：darkly/superhero/cyborg
window.geometry("700x500")

# ===== 你需要的所有组件 全都有，代码和原生tkinter几乎一样 =====
# 1. 输入框
entry = ttk.Entry(window, placeholder_text="请输入股票代码/合约名称", width=50)
entry.pack(pady=8)
# 2. 下拉选择器
combo = ttk.Combobox(window, values=['A股市场', '美股市场', '商品期货', '虚拟货币', '港股市场'])
combo.pack(pady=8)
combo.current(0)
# 3. 按钮组
btn_frame = ttk.Frame(window)
btn_frame.pack(pady=8)
ttk.Button(btn_frame, text="执行分析", bootstyle=SUCCESS).pack(side=LEFT, padx=5)
ttk.Button(btn_frame, text="保存策略", bootstyle=INFO).pack(side=LEFT, padx=5)
ttk.Button(btn_frame, text="重置参数", bootstyle=DANGER).pack(side=LEFT, padx=5)
# 4. 复选框
ttk.Checkbutton(window, text="启用K线指标(阴阳/双针/金叉)").pack(pady=2, anchor=W, padx=20)
ttk.Checkbutton(window, text="启用情绪共振策略").pack(pady=2, anchor=W, padx=20)
ttk.Checkbutton(window, text="启用格雷厄姆安全边际").pack(pady=2, anchor=W, padx=20)
# 5. 列表框 (带滚动条，分析结果展示)
list_box = ScrolledListbox(window, height=8)
list_box.pack(pady=8, fill=tk.BOTH, expand=YES, padx=20)
list_box.insert(tk.END, '【A股】茅台 - 金叉信号 持仓胜率87%')
list_box.insert(tk.END, '【美股】纳指 - 双针探底 趋势转强')
list_box.insert(tk.END, '【期货】原油 - 基本面利多 共振信号')

# 运行
window.mainloop()
```

import dearpygui.dearpygui as dpg

# 初始化
dpg.create_context()

        dpg.add_button(label="执行分析")
        dpg.add_button(label="保存策略")
        dpg.add_button(label="重置参数")
    # 4. 复选框
    dpg.add_checkbox(label="启用K线指标(阴阳/双针/金叉)")

    dpg.add_checkbox(label="启用格雷厄姆安全边际")
    # 5. 列表框
    dpg.add_listbox(('【A股】茅台 - 金叉信号 持仓胜率87%', '【美股】纳指 - 双针探底', '【期货】原油 - 利多共振'), num_items=3)

# 应用主题+显示
dpg.bind_theme("dark_theme")
dpg.setup_dearpygui()
dpg.show_viewport()
dpg.set_primary_window("main_window", True)
dpg.start_dearpygui()
dpg.destroy_context()

from PyQt5.QtWidgets import *

app = QApplication(sys.argv)

palette = QPalette()

css = '''

QListWidget{background-color:#444444; color:white; border:none; border-radius:6px;}
'''
window.setStyleSheet(css)

# 你的组件：和你原来的代码一模一样，不用改！
layout.addWidget(QLineEdit("请输入股票代码/合约名称"))

# ✅ 【打包EXE避坑指南】补充（重中之重，你一定会用到）
所有方案打包都用 **pyinstaller**，打包命令完全通用，给你2个万能命令，**解决99%的打包问题**，包括：打包后闪退、黑窗口、图标、体积过大。
> 所有方案都适用，复制即用！
### ✅ 打包命令1：基础版（无黑窗口+单文件）
```bash
pyinstaller -F -w -n 金融分析系统 你的脚本.py
```
- `-F`：打包成**单个EXE文件**，方便拷贝；
- `-w`：运行时**没有黑色控制台窗口**，纯GUI界面；
- `-n`：指定EXE文件名，不用默认的dist里的丑名字。

### ✅ 打包命令2：完美版（加图标+无黑窗+单文件+减小体积）
```bash
pyinstaller -F -w -i icon.ico -n 金融分析系统 --clean --noconsole 你的脚本.py
```
- `-i icon.ico`：给EXE加自定义图标（网上随便下载ICO格式图标就行）；
- `--clean`：清理缓存，减小体积；
- 打包后EXE在 `dist` 文件夹里，直接双击运行即可。
